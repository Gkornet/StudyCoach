# StudyCoach 📐

Persoonlijke wiskundedocent voor 3 VWO — gebouwd met Next.js + Claude AI.

## Setup

1. Clone deze repo
2. `npm install`
3. Maak `.env.local` aan met:
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```
4. `npm run dev` → open http://localhost:3000

## Vercel Deploy

1. Push naar GitHub
2. Importeer project op [vercel.com](https://vercel.com)
3. Voeg environment variable toe: `ANTHROPIC_API_KEY`
4. Deploy!
